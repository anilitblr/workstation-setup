# Alembic database migration tool

## Setup alembic

```bash
# To build a virtual environment
$ cd /path/to/your/project
$ virtualenv .venv

# install Alembic
$ source .venv/bin/activate
$ pip install alembic

# Initalize alembic
$ alembic init alembic
```

## create account table

```bash
alembic revision -m "create account table"
```
